# Hair Salon Landing Page

A modern **Hair Salon landing page** built with **HTML5 and CSS3**.

This project recreates a professional beauty salon website layout including a hero section, stylist team cards, promotional banner, lookbook gallery, services section, and promotional content.

## Features

* Hero banner with promotional content
* Stylist team profile cards
* Promotional haircut offer banner
* Lookbook image gallery
* Service highlights section
* Video promotion section
* Responsive layout
* Modern UI using Flexbox and CSS Grid

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Google Fonts

## Purpose

This project was built as part of my **front-end development portfolio** to practice modern layout design, image grids, and multi-section landing page structure.
